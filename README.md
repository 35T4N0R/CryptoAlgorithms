# CryptoAlgorithms

Implementation of simple cryptographic modules:

* Rail Fence
* 3 different matrix encryption
* Affine cipher
* Vigenere cipher

Stream ciphers:
* Linear Feedback Shift Register (LFSR) - generating pseudo-random numbers
* Synchronous Stream Cipher
* Ciphertext Autokey

Symmetric ciphers:
* Data Encryption Standard (DES)
